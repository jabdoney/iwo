Help/Information

This application generates an Income Withholding Order consistent with Form OMB 0970-0154, approved through September 30, 2023. The resulting document is in .docx format and has been specially formatted to process correctly when proposed through the Florida E-Filing Portal and routed to the ICMS Case application processing system. The rendering of the .docx file will appear to be formatted incorrectly. The user should not be concerned.

To avoid processing errors in ICMS, the .docx file generated by this application may not be edited directly. Users must make edits using the form controls contained within the application.

If edits are required after document creation, the user may import data from document by choosing the file using the control in the upper right corner of the screen, then clicking the "Import" button.
